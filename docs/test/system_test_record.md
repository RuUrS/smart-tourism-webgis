# 老君山智能旅游 WebGIS 系统测试记录

## 一、测试环境

| 项目       | 内容                   |
| ---------- | ---------------------- |
| 操作系统   | Windows 11             |
| 前端框架   | Vue 3 + Vite           |
| 后端框架   | FastAPI                |
| 数据库     | PostgreSQL + PostGIS   |
| 地图框架   | Leaflet + 高德地图瓦片 |
| 可视化框架 | ECharts                |
| 大模型     | 智谱 AI GLM-4.5-Flash  |
| 测试时间   | 2026年5月              |

## 二、功能测试表

| ---- | -------------- | -------------------------------- | -------------------------------------- | ------------------------------ | -------- |
