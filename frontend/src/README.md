# src

`src/` 目录是前端项目的核心源码目录，主要存放 Vue 应用入口文件、页面组件、路由配置、工具函数和全局样式。本项目基于 Vue 3 + Vite 开发，前端主要负责地图页面展示、用户入口、账号状态管理、接口请求和交互界面实现。

## 目录结构

```text
src
├─ assets/              # 前端静态资源目录，存放图片、图标、样式资源等
├─ components/          # Vue 组件目录，存放地图页面、登录入口、账号面板等组件
├─ router/              # 前端路由配置目录
│  └─ index.js          # 路由配置文件，用于管理页面访问路径和组件映射
├─ utils/               # 前端工具函数目录
│  └─ auth.js           # 登录状态、Token 保存、用户信息读取、用户记录保存等认证工具
├─ App.vue              # 前端根组件，负责控制系统入口、地图页面和账号状态栏展示
├─ main.js              # 前端应用入口文件，负责创建 Vue 应用并挂载到页面
├─ README.md            # 当前 src 目录说明文档
└─ style.css            # 全局样式文件，定义项目通用页面样式
```

## 文件夹说明

| 文件夹        | 作用                                                                        |
| ------------- | --------------------------------------------------------------------------- |
| `assets/`     | 存放前端静态资源，例如图片、图标、Logo、页面素材等。                        |
| `components/` | 存放 Vue 页面组件和功能组件，例如地图组件、登录入口组件、账号状态栏组件等。 |
| `router/`     | 存放前端路由配置，用于控制页面路径和组件之间的映射关系。                    |
| `utils/`      | 存放前端公共工具方法，例如登录状态管理、Token 读取、用户记录保存等。        |

## 主要文件说明

| 文件        | 作用                                                                                          |
| ----------- | --------------------------------------------------------------------------------------------- |
| `App.vue`   | 前端根组件，负责判断用户是否进入系统，并组合展示 `AuthEntry`、`MapView`、`AuthPanel` 等组件。 |
| `main.js`   | 前端入口文件，负责创建 Vue 应用、引入全局样式、挂载根组件。                                   |
| `style.css` | 全局样式文件，存放项目通用样式和页面基础样式。                                                |
| `README.md` | 当前目录说明文件，用于解释 `src/` 目录下各文件和文件夹的作用。                                |

## 说明

- 前端主要页面逻辑集中在 `components/MapView.vue` 中。
- 登录、注册、游客进入和账号状态相关功能由 `AuthEntry.vue`、`AuthPanel.vue` 和 `utils/auth.js` 配合完成。
- 前端接口请求建议统一使用 `/api/...` 路径，并通过 `vite.config.js` 中的代理转发到后端，避免在代码中写死 `localhost` 或局域网 IP。
- 全局样式放在 `style.css`，组件内部样式建议写在对应 `.vue` 文件中，便于维护。
