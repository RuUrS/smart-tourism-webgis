# routers

`routers/` 目录用于存放后端接口路由文件。项目后端基于 FastAPI 开发，不同功能模块按照文件进行拆分，便于维护和扩展。

## 文件说明

| 文件                | 作用                                                                                                             |
| ------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `auth.py`           | 用户认证与用户记录接口，包括用户注册、登录、Token 校验、获取当前用户、保存用户使用记录、查询用户历史记录等功能。 |
| `agent.py`          | 智能旅游助手接口，负责接入智谱 GLM 模型，实现旅游问答、行程规划、地图动作解析、多 Agent 调度等功能。             |
| `amap.py`           | 高德地图相关接口，主要用于在线地点搜索、行政区查询、路径规划、坐标处理等高德 Web 服务能力。                      |
| `boundary.py`       | 边界数据接口，用于查询研究区边界、景区边界、行政区范围等空间数据。                                               |
| `chat.py`           | 基础大模型问答接口，用于简单聊天问答或早期 AI 问答功能测试。                                                     |
| `crowd.py`          | 客流相关接口，用于提供模拟客流数据、景区客流态势展示等功能。                                                     |
| `dashboard.py`      | 数据大屏接口，用于提供首页数据统计、图表数据、客流概览、POI 统计等可视化数据。                                   |
| `knowledge.py`      | 旅游知识库接口，用于查询景区介绍、旅游提示、游览知识等结构化知识内容。                                           |
| `langchain_demo.py` | LangChain 示例接口，用于验证 LangChain 调用流程或保留后续扩展智能体能力。                                        |
| `poi.py`            | POI 查询接口，用于查询景区点位、附近搜索结果、兴趣点详情等空间点数据。                                           |
| `recommend.py`      | 推荐路线接口，用于根据景点、天气、交通、客流等信息生成或返回推荐游览路线。                                       |
| `road.py`           | 路网数据接口，用于加载基础路网、主路、景区道路、步道等道路空间数据。                                             |
| `route.py`          | 路线规划接口，用于处理本地路线规划、路径结果查询、路线 GeoJSON 返回等功能。                                      |
| `traffic.py`        | 交通态势接口，用于提供周边交通状态、道路拥堵情况或模拟交通数据。                                                 |
| `weather.py`        | 天气查询接口，用于获取研究区或指定城市的实时天气信息，并为路线推荐和游览提示提供天气参考。                       |

## 使用说明

这些路由文件会在 `backend/main.py` 中统一注册到 FastAPI 应用中。后端启动后，可通过 FastAPI 自动生成的接口文档查看具体接口路径、请求参数和响应结果。

本地接口文档地址：

```text
http://127.0.0.1:8000/docs
```

局域网访问时，可将 `127.0.0.1` 替换为后端所在电脑的局域网 IP。

## 说明

- 本目录只存放接口路由层代码。
- 数据库连接、业务处理、工具函数等逻辑应尽量放在 `database.py`、`services/` 或其他业务模块中。
- 真实环境变量、API Key、数据库密码等敏感信息不应写入本目录代码中，应统一放在 `.env` 文件中，并通过 `.gitignore` 排除上传。
